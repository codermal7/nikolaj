ABOUT US

We are second year Btech CSE students of Bennett University who developed this project for our AI Hackathon which was held on 12-13th November. What interested us most was how we can use state provided crime data in order to have an efficient police resource distribution which would help in reducing crimes. The idea behind this project is to make use of available crime dataset in order to create optimal routes with respect to crime severity, time stamp and availablility of on duty police officers.


As user input, our program requires the following:
(1) the city that the plot is for
(2) the number of precincts
(3) average number of officers per precinct
(4) the timeframe of the shift
